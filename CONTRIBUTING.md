# Contributing

1. Fork + branch
2. `npm i`
3. `npm run dev`
4. `npm run lint && npm run typecheck`
5. PR s popisem a screenshoty

## Commity
Preferuj conventional commits: `feat:`, `fix:`, `chore:`, `docs:`, `refactor:`, `build:`.

## Release
Maintainer vytvoří tag `vX.Y.Z` → GitHub Actions vytvoří release a přiloží `dist/`.
